pybind11 Library
----------------

This directory contains a copy of the pybind11 header-only library.
This is currently synced with pybind11 version 2.10.4 (16 Mar 2023).

pybind11 is distributed under a BSD-style license that can be found in the
LICENSE file in this directory.

pybind11 was created by Wenzel Jakob (http://rgl.epfl.ch/people/wjakob),
with significant features and/or improvements by many others, and can be
downloaded from https://github.com/pybind/pybind11 .
