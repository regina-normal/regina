Regina Engine Directory
-----------------------

This directory tree contains the C++ sources for the main calculation engine.
This is where you will find all the "real mathematics".

The doxygen support files for generating the API documentation (which were
located here in Regina ≤ 7.x) are now located in apidocs/ in the main source
tree.
