Building Regina from Source

Tip: See the page on building Regina on different systems, which offers advice tailored for your particular type of machine.

Regina's build scripts should detect most aspects of your system configuration, and give you useful error messages if something looks out of place. If Regina does not build out of the box for you, please mail us (including your CMakeCache.txt if possible) so that we can update the scripts to work with systems like yours in the next release.

In summary, to build Regina:

  • Make sure you have installed all the necessary tools and libraries as listed in the system requirements table below.
  • Download the source archive regina-4.90.tar.gz from the Regina download site.
  • Unpack this source archive, which will expand into a directory regina-4.90/ on your machine.

    example:~$ tar -zxf regina-4.90.tar.gz

  • Create a build subdirectory, such as regina-4.90/build/.

    example:~$ cd regina-4.90
    example:~/regina-4.90$ mkdir build

  • Move into the build directory, and then run “cmake ..”. This will search for the necessary tools and libraries on your system, and configure your build accordingly.

    example:~/regina-4.90$ cd build
    example:~/regina-4.90/build$ cmake ..

    You will almost certainly need to pass additional arguments to cmake. Please visit this page for details.
  • If cmake succeeds, run make to build Regina. Then, as root, run make install to install Regina on your system.

    example:~/regina-4.90/build$ make
    example:~/regina-4.90/build$ sudo make install

That's all there is to it. If you are having trouble building or running Regina, please check out the troubleshooting page and/or mail us at regina-user@lists.sourceforge.net for assistance.

System Requirements

The following table lists the tools and libraries that you will need to build Regina.

It is highly recommended that you download ready-made packages of these tools and libraries. This is the best way to provide standard installations in standard locations that Regina can easily find and use. All of these tools and libraries are included in almost every major GNU/Linux distribution.
